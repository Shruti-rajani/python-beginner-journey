# Week 5 – Quiz Game (Python)

This project is part of my learning journey where I built a simple **quiz game in Python** with scoring and a replay feature.

---

## Features

*  Multiple quiz questions
*  Answer validation (case-insensitive)
*  Score tracking
*  Replay option after finishing the quiz
*  Displays correct answers if the user is wrong

---

##  How It Works

* A dictionary stores questions and their correct answers
* The program loops through each question
* User inputs their answer
* Score increases for each correct answer
* At the end, the total score is displayed
* User can choose to play again
